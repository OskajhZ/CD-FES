# README

Codes for the proposed system:

1. ```feature_define.py```: CD-FES definition, run to compute CD-FES
2. ```model_define.py```: Classifiers definition (including two-level fusion LSTM and other classifiers)
3. ```main.py```: main entry of the proposed system, run to start training and testing

Codes for reproductions:

1. ```reproduction.py```: Reproductions of Shen2017 and Cai2018, and feature comparison excluding DE, CD-AED and CD-FES. Rely on ```model_define.py```.
2. ```TiSc_replication.py```: Reproduction of Time Scale Network
3. ```SFCSAN_Replication```: Including the reproduction of SFCSAN, and feature comparision of Differential Entropy (DE)
4. ```EEGNet_replication.py```
