# Readme

This repository is for the following article:

Lixian Zhu, Xiangnan Zhang, Ranqi Lu, Jingyu Liu, Fuze Tian, Juan Wang, Jingxin Liu, and Bin Hu, "Frequency-Domain Entropy Sequence as a Dynamic EEG Representation for Depression Recognition"

In ```SourceFiles```: available codes for the proposed CD-FES computing and two-level fusion LSTM network, feature comparison, and reproductions. And FES validation on SEED and DEAP dataset as well.

In ```ExperimentLogs```: original logs corresponding to the results shown in the article.
